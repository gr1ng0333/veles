# Search (`search.py`)

## Назначение
Простой инструмент для веб-поиска. Позволяет агенту Ouroboros искать информацию в интернете напрямую через *OpenAI Responses API* (бывш. ChatGPT с подключенным Web Search). Модель выступает агрегатором результатов (получает поисковые запросы, серфит и собирает итоговый результат). 

Требует установленной переменной `OPENAI_API_KEY`.

## Доступные инструменты (Tools)

### `web_search`
- **Описание**: Выполнить поисковый запрос в интернет. Возвращает ответ (сводку) собранную моделью OpenAI (по умолчанию `gpt-5`, можно переопределить через `OUROBOROS_WEBSEARCH_MODEL`).
- **Параметры**:
  - `query` (str, required): Поисковый запрос.
