# Browser Tools (`browser.py`)

## Назначение
Мощный модуль автоматизации браузера (под управлением Playwright), позволяющий агенту визуально или программно взаимодействовать с веб-страницами (без API). Создан с учётом многопоточности (greenlets Playwright).

Состояние браузера (подключения, вкладки, скриншоты) хранится в `ToolContext.browser_state`. Жизненный цикл браузера привязан к текущей задаче. На уровне модуля (`_pw_instance`) хранится глобальный Playwright для переиспользования между задачами (пока не произойдет смена треда). Модуль содержит механизмы авто-восстановления (kill процессов Chromium, сброс состояний `sys.modules`) при ошибках из-за greenlet-потоков.

Зависимости: `playwright`, `playwright_stealth` (опционально). При отсутствии Playwright инструмент пытается установить его автоматически (вместе с зависимостями `chromium`).

## Доступные инструменты (Tools)

### `browse_page`
- **Описание**: Открывает URL в headless («безголовом») браузере с разрешением 1920x1080.
- **Выходные форматы** (`output`):
  - `text`: Обычный внутренний текст тела страницы (`inner_text`).
  - `html`: Сырой код страницы.
  - `markdown`: Специфичный JS-скрипт (`_MARKDOWN_JS`) рендерит элементы дерева (H1, DIV, A, LI) в структурированный Markdown.
  - `screenshot`: Делает скриншот вкладки и кодирует в PNG Base64. (Может быть отправлен владельцу через `send_photo`).
- **Параметры**:
  - `url` (str, required): Адрес для открытия.
  - `output` (str): Формат (text, html, markdown, screenshot). По умолчанию: `text`.
  - `wait_for` (str): CSS-селектор элемента, которого нужно дождаться перед извлечением контента.
  - `timeout` (int): Тайм-аут загрузки (ms). По умолч: `30000` (30 сек).
- **Схема работы**: Окно браузера (`page.goto`) персистирует между вызовами в пределах одной задачи (если агент не закончил выполнение).

### `browser_action`
- **Описание**: Выполнение конкретного действия (action) на текущей (уже открытой) открытой странице браузера. 
- **Поддерживаемые Action**:
  - `click`: Клик по `selector`.
  - `fill`: Ввод текста `value` в `selector`.
  - `select`: Выбор поля из Dropdown списка (`selector` + `value`).
  - `screenshot`: Создание Base64 PNG, обновляя `__last_screenshot__` в контексте.
  - `evaluate`: Исполнение произвольного JavaScript-кода (`value`) и возврат строки результата (до 20к символов).
  - `scroll`: Скролл окна (value: `up`, `down`, `top`, `bottom`).
- **Параметры**:
  - `action` (str, enum, required): Действие на странице.
  - `selector` (str): CSS-селектор элемента.
  - `value` (str): Доп. значение (клик/ввод, код JS, направление скролла).
  - `timeout` (int): Ожидание выполнения (ms). По умолч: `5000`.

*При крашах из-за потоков, библиотека автоматически выполняет очистку (force kill chrome & reset playwright state) и повторяет действие.*
