# Shell Tools (`shell.py`)

## Назначение
Набор инструментов для выполнения shell-команд и делегирования задач редактирования кода в `Claude Code CLI`. Инструменты помечены флагом `is_code_tool=True` для приоритетной обработки и контроля тайм-аутов.

## Доступные инструменты (Tools)

### `run_shell`
- **Описание**: Выполняет произвольную shell-команду (переданную как массив аргументов) внутри репозитория. Имеет встроенную защиту от тайм-аутов (120 секунд) и ограничивает вывод STDOUT+STDERR (обрезает до 50 000 символов: 25к в начале + 25к в конце). Если команда ошибочно передана как строка вместо массива, пытается разобрать её через `shlex.split`, логируя предупреждение в `events.jsonl` (папка `logs/`).
- **Параметры**:
  - `cmd` (array of strings, required): Команда и её аргументы.
  - `cwd` (str): Относительная рабочая директория (от корня репозитория) для выполнения команды.

### `claude_code_edit`
- **Описание**: Делегирует сложные рефакторинги, исправления багов и многофайловые правки утилите `Claude Code CLI`. Инструмент берет на себя блокировку Git (`git lock`), гарантирует переключение в рабочую ветку (`branch_dev`) и запускает процесс с ограниченными правами. При завершении парсит JSON-ответ от Claude, автоматически подсчитывает затраты на LLM в `EVENTS` очередь и проверяет наличие незакоммиченных изменений (`git status`, выдавая предупреждение агенту).
- **Особенности реализации**:
  - Требует наличие переменной окружения `ANTHROPIC_API_KEY`.
  - Запускается с флагами `--output-format json`, `--max-turns 12` и набором разрешенных тулзов `Read,Edit,Grep,Glob`.
  - Учитывает разные версии флага разрешений Claude (`--permission-mode bypassPermissions` с откатом к устаревшему `--dangerously-skip-permissions`).
  - Тайм-аут: 300 секунд.
- **Параметры**:
  - `prompt` (str, required): Подробный промпт/описание того, что Claude должна изменить. Автоматически дополняется инструкциями для строгой работы внутри рабочей директории без коммитов (`STRICT: Only modify files inside...`).
  - `cwd` (str): Рабочая директория, на которую будут ограничены правки.
